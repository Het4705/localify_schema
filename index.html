Localify Database Schema Documentation

Company Schema

Field Type Description
company_id ObjectId Company's unique Id (auto-generated by MongoDB)
name String Company name
location Object Company's location details

Location Schema

Field Type Description
city String City of the company
state String State of the company
country String Country of the company
pincode String Pincode of the company
tagline String Tagline of the company
brief_info String Brief information about the company
owner_id ObjectId Reference to the owner of the company
email String Contact email for the company
logo String URL or path to the company's logo
banner_image String URL or path to the banner image

Product Schema

Field Type Description
product_id ObjectId Product's unique Id (auto-generated by MongoDB)
company_id ObjectId Reference to the company that owns the product
name String Name of the product
category String Category of the product
description String Description of the product
brand String Brand of the product
video_url String URL to a video related to the product (optional)
primary_image String URL or path to the primary image of the product
product_images Array of Strings URLs or paths to additional product images (optional)
price Number Price of the product
stock Number Available stock quantity of the product
rating Number Rating of the product (optional)
reviews Array of Objects Reviews by customers including review details and images (optional)

Review Schema

  • review_id: ObjectId (Automatically generated)
  • customer_id: ObjectId (Reference to the customer)
  • review: String (Review text)
  • images: Array of Strings (URLs or paths to images, optional)
  • upvotes: Number (Optional)

Order Schema

Field Type Description
Order_id ObjectId Order's unique Id (auto-generated by MongoDB)
customer_id ObjectId Reference to the customer who placed the order
company_id ObjectId Reference to the company from which the order was placed
products Array of Objects Product's detail

product Schema

Field Type Description
quantity Number Product quantity
product_id ObjectId Reference to the Product
total Number Total amount of the order
status String Current status of the order (pending, delivered, in process, cancelled, returned)

Customer Schema

Field Type Description
customer_id ObjectId Customer's unique Id (auto-generated by MongoDB)
name String Name of the customer
email String Email address of the customer (unique)
password String Password for customer account
location Object Customer's location details

Location Schema

Field Type Description
city String City of the customer
state String State of the customer
country String Country of the customer
pincode String Pincode of the customer
phone String Contact phone number of the customer
created_at Date Date when customer account was created
last_login Date Date of last login for the customer
company_id ObjectId Reference to the customer's associated company

Owner Schema

Field Type Description
owner_id ObjectId Owner's unique Id (auto-generated)
email String Email address of the owner (unique)
password String Password for owner account
name String Name of the owner
dob Date Date of birth of the owner
details String Additional details about the owner (optional)
contact_number String Contact number of the owner
image String URL or path to the owner's profile image (optional)
timestamps Date Timestamps for creation and last update of the owner record
